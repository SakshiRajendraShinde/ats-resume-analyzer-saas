# Automated Resume Screening Tool

## Overview
This project simulates an Applicant Tracking System (ATS) used by companies to screen resumes.

## Problem
Manual resume screening is time-consuming and inefficient.

## Solution
This tool automatically:
- Extracts resume text
- Matches with job description
- Scores and ranks candidates

## Features
- PDF & DOCX parsing
- TF-IDF similarity
- Resume ranking
- CSV report generation
